# 🚀 AI Cold Email Generator

Generate highly personalized, professional cold emails in seconds using AI.

This project helps job seekers create recruiter-ready cold emails, LinkedIn messages, subject lines, and follow-up emails based on their resume and a job description.

---

## ✨ Features

- 🤖 AI-powered cold email generation
- 📄 Resume & Job Description analysis
- ✉️ Professional cold emails
- 💼 LinkedIn connection requests
- 💬 LinkedIn follow-up messages
- 📧 Follow-up email generation
- 📝 AI-generated email subject lines
- ⚡ Fast and responsive UI
- 🔐 User Authentication
- 📚 Email History
- 🎨 Modern Responsive Design

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/ai-cold-email-generator.git
```

```bash
cd ai-cold-email-generator
```

---

### Install Frontend

```bash
cd client
npm install
npm run dev
```

---

### Install Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

GROQ_API_KEY=your_groq_api_key
```

## 🚀 How It Works

1. Upload or paste your Resume.
2. Paste the Job Description.
3. AI analyzes your profile.
4. Generates:
   - Cold Email
   - Subject Line
   - LinkedIn Connection Request
   - LinkedIn Follow-up
   - Follow-up Email
5. Copy and send with one click.

---

## 🎯 Why This Project?

Applying to jobs manually takes time.

This application uses AI to generate personalized outreach messages that save time while maintaining a professional tone.

Instead of writing every email from scratch, users can generate tailored messages within seconds.
